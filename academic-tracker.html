Academic Progress Tracker

This Java-based console application is built to help students calculate grades and view progress for each subject. It is focused on classroom learning and exam preparation.

The current version supports input of subject marks, GPA calculation, and grade reports through a command-line interface.

Next steps include improving data input validation, adding monthly progress summaries, and exporting results to a simple text report.
