Study Notes Organizer

This project is a notes organizer built with JavaScript to help students store study points and access them quickly. It uses browser local storage so notes remain available between sessions.

It is designed to support categories, simple editing, and easy retrieval of study material while I continue learning DOM manipulation and front-end logic.

Next, I plan to add search functionality, note tags, and better mobile usability.
