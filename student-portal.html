Student Portal Prototype

This project is a student portal prototype designed to help learners organize study schedules, access course notes, and track assignment deadlines in one place.

The current version is built with HTML, CSS, and JavaScript. It includes a responsive layout, a notes section, and a schedule planner UI that can be extended to support user accounts.

It is still under development, and my next steps are to add interactive task cards, database support, and a working note upload/download feature.
