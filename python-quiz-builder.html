Python Quiz Builder

This Python project generates practice quizzes for self-study and review. It is built as a command-line tool to help me learn Python control flow, input handling, and scoring logic.

The program currently supports multiple-choice questions, score tracking, and a summary of results at the end of each quiz session.

Future updates will include question categories, timing features, and improved question generation for programming concepts.
